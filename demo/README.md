# ADSL Demo

This folder contains a professional Streamlit demo of the ADSL platform.

## Quick Start

```bash
pip install streamlit pandas
streamlit run demo/streamlit_demo.py
```

## What the Demo Shows

- Scenario comparison (Baseline vs Contested)
- Key risk metrics and bottlenecks
- Actionable recommendations
- Clean interface suitable for enterprise and defense stakeholders

## Purpose

This demo is designed to quickly communicate ADSL's value to technical and non-technical stakeholders, including Palantir teams.
